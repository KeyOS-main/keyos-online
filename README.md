## What can the KeyOS environment do?

  The Online KeyOS environment can do multiple things, including:
  
  * Alpine Linux *(via JS/Linux!)*
  * Basic gaming (Anything HTML5/js)
  * Browse the web!
  * Customize itself!
  * Get new apps with the Key Store!
  * x86 Emulation!
